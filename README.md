# Classification and Prediction of Product Attributes for Etsy Marketplace

## Overview
Built a machine learning pipeline to classify and predict product attributes 
from Etsy marketplace listings — automating categorisation at scale using 
NLP and supervised learning techniques.

## Dataset
- 245,485 rows of Etsy product listing data
- Features include product titles, descriptions, and categorical attributes

## Methods & Techniques
- **Text Preprocessing** — tokenisation, stopword removal, feature engineering
- **TF-IDF Vectorisation** — converted raw text into numerical feature vectors
- **Naive Bayes Classifier** — trained and evaluated for multi-class classification
- **Model Evaluation** — F1 Score of 0.444, representing an 11% improvement over baseline

## Key Outcomes
- Achieved consistent improvement over baseline across multiple product categories
- Demonstrated scalable NLP pipeline applicable to e-commerce attribute tagging
- Identified feature engineering as the highest-leverage lever for further improvement

## Tools & Libraries
Python, Jupyter Notebook, Scikit-learn, Pandas, NumPy, NLTK
